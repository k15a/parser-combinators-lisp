# Comments

## Line Comments

```lisp
; This is a line comment
```

## Block Comments

```lisp
#| This is a block comment |#
#| It can span
multiple lines |#
```

# Lists

```lisp
(1 2 3 4 5)
("Hello" "World")
```

# Function Declarations

```lisp
(defun add (a b c) (+ a b c))
```

# Function Calls

```lisp
(add 1 2 3)
```

# Inline Functions

```lisp
(lambda (arg) (+ arg 1))
```
